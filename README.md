🎯 Project Overview

The Cupcake Store Website showcases a clean and interactive UI for an online bakery.
It focuses on delivering a delightful user experience through modern design trends, responsive layouts, and smooth animations.

✨ Features
🍰 Interactive Hero Section with image switching
🧁 Beautiful Cupcake Gallery
📖 Clean About Section
📱 Fully Responsive Design (Mobile, Tablet, Desktop)
🎨 Modern UI (Glassmorphism + animations)
🔘 Action buttons (Live Demo / GitHub)
🧭 Smooth navigation & user-friendly layout

🛠️ Technologies Used
HTML5
CSS3
Bootstrap 5
SASS (SCSS)

⚙️ Installation & Setup
Clone the repository:
git clone https://github.com/salmaassem-eng/Cupcakes-website.git
Open the project folder:
cd cupcake-website
Run with Live Server or open index.html in your browser.
